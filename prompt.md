Detail siswa
buatkan saya ui dashboard mobile aplikasi penanganan pembayaran dengan desain dan elemen seperti ini. halaman tersebut berisi detail siswa (nama siswa, id yayasan, no.hp, lembaga, kelas, asrama, kamar asrama, madin, kelas madin), detail pembayaran membuka popup, total tunggakan, saldo saat ini, jenis penanganan (tombol tindakan ikon chat/telepon yang nantinya muncul modal (tombol kirim pesan dengan tunggakan & tombol kirim pesan kosong), catatan tindakan, tombol simpan), histori jenis penanganan, tombol penanganan selesai (muncul modal dengan field hasil (lunas, isi saldo, cicilan, tidak ada respon, hp tidak aktif) catatan, rating), tombol rekomendasi (muncul modal dengan judul Buat Rekomendasi, field tanggal rekomendasi, tombol kirim link pernyataan rekomendasi), daftar penanganan sebelumnya.
